```
┌─────────────────────────────────┐
│  ←                        ⋮     │
├─────────────────────────────────┤
│  ┌───────────────────────────┐  │
│  │                           │  │
│  │        [상품 이미지]       │  │
│  │                           │  │
│  └───────────────────────────┘  │
│   ●  ○  ○  (이미지 인디케이터)   │
├─────────────────────────────────┤
│  [판매중] 가구 > 침대/매트리스    │
│  침대 프레임 퀸사이즈             │
│  150,000원                      │
│  등록일: 2025-06-01             │
├─────────────────────────────────┤
│  판매자 정보                     │
│  👤 김민준  ⭐ 4.8 (리뷰 23개)   │
├─────────────────────────────────┤
│  상품 설명                       │
│  2년 사용, 상태 양호합니다.       │
│  이사 때문에 급처합니다...        │
├─────────────────────────────────┤
│  ♥ 찜하기(7)    [채팅하기]       │
│              [예약하기]          │
└─────────────────────────────────┘
```

**연결 API**: `GET /products/{productId}`
**동작**:
- [채팅하기] → 비로그인 시 로그인 유도 → `POST /chat-rooms` (또는 기존 채팅방 진입)
- [예약하기] → `POST /products/{productId}/reserve`
  - 성공: 채팅방으로 이동
  - 실패(409): "이미 예약된 상품입니다" 토스트
- [찜하기] → `POST /products/{productId}/likes`
- 판매자 본인: [수정] [삭제] 버튼 표시, 예약/채팅 버튼 숨김

- `409` → 이미 리뷰 작성 완료 시 "이미 작성한 리뷰가 있습니다." 토스트 후 이전 화면 이동

## 화면별 API 매핑 요약 (전체 13개)

| 화면 | 주요 API 호출 |
|------|--------------|
| S01 홈 | `GET /products` |
| S02 검색 | `GET /search/popular`, `GET /products/search` |
| S03 상품 상세 | `GET /products/{id}`, `POST /products/{id}/likes`, `POST /products/{id}/reserve` |
| S04 상품 등록 | `GET /categories`, `POST /products` |
| S05 상품 수정 | `GET /products/{id}`, `PATCH /products/{id}` |
| S06 채팅방 목록 | `GET /chat-rooms` |
| S07 채팅방 상세 | `GET /chat-rooms/{id}/messages` + WebSocket STOMP |
| S08 마이페이지 | `GET /members/me`, `POST /auth/logout` |
| S09 내 판매 목록 | `GET /members/me/products`, `PATCH /trades/{id}/status` |
| S10 내 찜 목록 | `GET /members/me/likes`, `DELETE /products/{id}/likes` |
| S11 로그인 | `POST /auth/login` |
| S12 회원가입 | `POST /auth/signup` |
| S13 리뷰 작성 | `POST /trades/{tradeId}/reviews` |

**컬러 팔레트**:
- Primary: `#FF6F0F` (당근 오렌지 계열)
- Background: `#FFFFFF`
- Surface: `#F5F5F5`
- Text Primary: `#1A1A1A`
- Text Secondary: `#767676`
- Error: `#E53E3E`

**타이포그래피**:
- 제목: 18px Bold
- 본문: 14px Regular
- 캡션: 12px Regular

1. 상품 카드 (목록, 검색에서 반복 사용)
